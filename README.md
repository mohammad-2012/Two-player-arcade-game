# 🎮 Arcade Duo

Two classic games – **Snake** & **2048** – in one minimalist arcade.

---

## ✨ Features
- 🌗 Dark/Light theme toggle
- 📱 Fully responsive (mobile + desktop)
- ⌨️ Keyboard & touch controls
- 🏆 Score & best score tracking
- 🎯 On-screen D-Pad for Snake

---

## 🛠️ Built With
- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Vazirmatn)
- No frameworks – pure vanilla
